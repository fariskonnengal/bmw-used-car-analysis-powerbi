# BMW Used Car Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes BMW used car sales data to identify pricing trends and factors affecting resale value.

## Tools Used
- Power BI
- Power Query
- Excel

## Dashboard Preview

![Dashboard](BMW_Dashboard.png)

## Key Insights

- Newer BMW models command higher average prices.
- Hybrid vehicles have the highest average resale value.
- Premium models such as X7, 8 Series, and M5 are the most expensive.
- Semi-Automatic transmission vehicles have higher average prices.
- Higher mileage generally reduces resale value.
